# Phoenix Janta Party App

Official Phoenix Janta Party PWA App.

## Features

- Professional mobile app design
- Install App button
- Offline support
- Motivational quotes
- Join form
- Share app button
- Responsive design
- GitHub Pages ready

## GitHub Pages Setup

1. Upload all files to GitHub.
2. Go to Settings.
3. Open Pages.
4. Select branch: main.
5. Select folder: /root.
6. Click Save.

Your app will be live at:

https://yourusername.github.io/phoenix-janta-party-app/
